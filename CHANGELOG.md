# Change Log

## 0.0.1

### Added

- View and edit office documents, spreadsheets, presentations, and PDFs attached to email messages directly in Outlook using ONLYOFFICE Docs.
- Create new DOCX, XLSX, and PPTX files from the compose window; the created file opens in the editor immediately and is added as an email attachment.
- Settings panel to configure ONLYOFFICE Docs server URL and JWT secret key; settings are stored in roaming settings.
